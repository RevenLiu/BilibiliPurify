# Bilibili Purify 2.0 / Bilibili 纯粹化 2.0

一个应用于 **Bilibili** 平台的篡改猴脚本（Tampermonkey UserScript）。

以一种粗暴而直接的方式，抵抗商业化平台对人类注意力与情绪的利用。
通过削弱算法、广告与信息噪音的影响，让你重新获得对自己使用体验与思考节奏的控制权。

## ✅ 功能

- 自动将主页直接跳转到搜索页，避免无意识刷屏和目的性视频推荐，拒绝算法。

- 隐藏主站、直播区的广告条、活动推送、无关入口与几乎所有推荐列表。
  保留核心功能，去除视觉噪音。

- 移除评论区。
  铲除了情绪裹挟扎根的土壤。

- 移除几乎所有量化指标。包括但不限于播放量、点赞数、投币数、收藏数、粉丝数、获赞数等数值。
  抹除基于情绪表达的社会证明。

- 隐藏了直播页当中绝大部分的可被用于引导消费、心理暗示、群体认同感操控的内容，以及会影响体验的礼物、广告等内容。
  如等级图标、特殊的聊天背景颜色等。

- 默认模糊搜索页中的视频封面，仅在主动选择时才显示。
  限制封面的第一视觉引诱。

- 自动隐藏搜索页中可能的因分析算法而推荐出现的视频。
  让搜索回归本意。

## 🧾 2.0 版本变更说明
> "如果不彻底消除死亡，禁枪便毫无意义。"

以当下的眼光来看，这在复杂的现实中是有失偏颇的。但幸运的是，在这里它成立。

- 1.x 版本中"评论区锁"与"评论区净化"功能被移除。

- 从根本上删除了绝大部分的互动渠道，不再有评论区，亦不再有回复和点赞。将责任全部还给你。

- 隐藏了绝大部分数值显示，视频、番剧或创作者个体，不再拥有数据带来的隐性背书，而都由你自己判断其优劣价值。

- 处理了若干具有引诱性的交互逻辑。

- 弹幕因其在部分情况中不可或缺的工具属性而被保留，但移除其默认合法性，默认关闭弹幕，且不记忆开关状态。

通过移除绝大部分的社交属性本身，显著地消除了情绪裹挟、群体极化、社会证明、无意义评价等问题。

***❗注意： 如果你绝对需要社交属性，可自行获取并回滚至 [v1.4.15](https://github.com/RevenLiu/BilibiliPurify/blob/1ed35e37fa5d74ba2f5398144af2dc4b16c98de9/BilibiliPurify.js)。但请注意，1.x 版本将不会被继续维护。***
